# Lore Intro Relay (GitHub Pages)

Статическая страница для плагина **Helix** `zzz_lore_intro`: встраивание YouTube в Garry's Mod без типичного Error 153 у `asset://html`.

## Быстрый старт

1. Создайте репозиторий (можно публичный, только под этот relay).
2. В **корень** репозитория добавьте:
   - `index.html`
   - `.nojekyll` (пустой файл)
3. **Settings → Pages** → Source: ветка `main`, папка **`/ (root)`** (или положите файлы в `docs/` и выберите `/docs`).
4. Дождитесь адреса вида `https://<user>.github.io/<repo>/`.
5. В игре в конфиге Helix укажите **`loreIntroRelayURL`** = этот URL **без** `?` в конце.  
   Ссылку на ролик настраиваете в **`loreIntroVideoID`** на сервере, не в HTML.

## Проверка

Откройте в браузере:

`https://<user>.github.io/<repo>/?v=dQw4w9WgXcQ`

Должен открыться полноэкранный плеер. Без `?v=` отображается эта документация.

## Параметры

| Параметр | Описание |
|----------|----------|
| `v` | YouTube video id (обязателен для режима плеера) |
| `q` | `vq`: `highres`, `hd2160`, … |
| `m` | `1` → `youtube.com`, иначе `youtube-nocookie.com` |
| `muted` | `1` без звука; `0` или отсутствует — со звуком |
| `crop` | `0` без обрезки оверлея; иначе обрезка краёв iframe |

Параметры `v`, `q`, `m`, `muted`, `crop` к URL дописывает клиентский плагин автоматически.
