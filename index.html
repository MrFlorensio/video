Lore Intro Relay

Статическая страница для встраивания YouTube в Garry's Mod (Helix, плагин zzz_lore_intro). Игра открывает этот URL с параметрами — без своего asset:// меньше Error 153.

Деплой на GitHub Pages

  1. Создайте репозиторий и положите в корень файлы index.html и .nojekyll из этой папки.
  2. Settings → Pages → Branch main, folder / (root) (или /docs).
  3. Скопируйте выданный URL в конфиг сервера: loreIntroRelayURL — без ? в конце.
  4. ID ролика задаётся в игре: loreIntroVideoID, не на этой странице.

Параметры запроса

vОбязателен для плеера. YouTube video id.
qКачество vq: highres, hd2160, … По умолчанию highres.
m1 — домен youtube.com; иначе youtube-nocookie.com.
muted1 — без звука. 0 или нет — со звуком (если браузер разрешит).
crop0 — без обрезки. Иначе — сдвиг/масштаб iframe, чтобы убрать часть оверлея YouTube.
